### 说明
AI 练手项目。 主要是用 `claude code` 和 `codex` 。右侧Release有打包好的APP。


### 列表
- [Mac性能监控](mac-monitor)： macOS 菜单栏监控工具。展示CPU和内存使用占比，以及网络上传下载速度。
![](./mac-monitor/demo.png)



- [浏览器视频播放器](browser-video-player) ：美化视频播放器，直接引用 css 和 js , 添加元素 `<video class="video-player" src=""></video>` 即可。
![](./browser-video-player/demo.png)

- [视频播放器](browser-video-player) : 同浏览器视频播放器，做了一层APP封装。


- [图片转换](image-tool) ：支持格式转换、压缩、缩放。
![](./image-tool/demo.png)
